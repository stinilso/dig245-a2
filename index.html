The above images was found by Googling an image of a whale shark and inspecting the page to find the image source, which was then copied into the HTML code of this web page. Below are Instructions for viewing the HTML source (using Google Chrome for a browser).


  1. Open up Google Chrome and navigate to the webpage you want to inspect.
  2. If you are using a computer with a touchpad, use two fingers to click anywhere on the webpage. A menu with several options will appear where you tapped.
  3. Click the "Inspect" option. A pane detailing the webpage's code should appear on the right side of the browser.
  4. As you navigate the webpage(s), the HTML source display will change accordingly to reflect what you are viewing. When you are done viewing the source code of a page, click the "x" in the upper right hand corner of the pane to close it.

All of these steps enable a user to view and analyze the source code. You can think of it like "looking up the recipe" to understand a meal you want to learn how to make.
